Server Documentation

Class: Server

The Server class provides functionality for handling server operations.

Constructor

      Server(
        int domain,       // AF_UNIX for local communication, AF_INET for internet domain
        int type,         // SOCK_STREAM, SOCK_DGRAM or SOCK_SEQPACKET (SOCK_STREAM is norm)
        int protocol,     // Standard is 0
        int port,         // Desired PORT ex 80
        // u_long interface, // Address to bind to.
        int backlog       // Number of pending connections in the backlog
      );

Public Methods

  • void start() - Starts the server.
  • void deploy() - Deploys the server.
  • void add_route(std::string route, ROUTE_HANDLER*) - Adds a route to the server.
  • BindingSocket *get_socket() - Returns the server socket.

Private Methods

  • int accept_client() - Accepts a client connection.
  • void handle_client(int t_client) - Handles a client connection.

Type Definitions

  • URL_PARAMETERS - Type alias for an unordered map of URL parameters.
  • ROUTE_HANDLER - Type alias for the route handler function signature.
  • REQUEST - Type alias for an unordered map of request parameters.
