BaseSocket Documentation

Class: BaseSocket

The BaseSocket class provides a basic socket implementation.

Constructor

      BaseSocket(
      int domain,     // AF_UNIX for local, AF_INET for internet domain
      int type,       // SOCK_STREAM, SOCK_DGRAM or SOCK_SEQPACKET
      int protocol,   // Standard is 0
      int port        // Desired PORT ex 80
      // u_long interface // ??
      );

Public Methods

  • int connect_to_network(int t_sck, struct sockaddr_in6 t_addr) - Virtual function that needs to be fulfilled by child class. Connects to the network.
  • void check_connection(int t_socket_fd, char* err_msg) - Checks the socket connection.
  • const struct sockaddr_in6& get_address() - Returns the address of the socket.
  • const int& get_sock() - Returns the socket file descriptor.

Class: BindingSocket

The BindingSocket class extends the BaseSocket class and provides functionality for binding and listening to a socket.

Constructor

      BindingSocket(
      int domain,     // AF_UNIX for local communication, AF_INET for internet domain
      int type,       // SOCK_STREAM, SOCK_DGRAM or SOCK_SEQPACKET (SOCK_STREAM is norm)
      int protocol,   // Standard is 0
      int port,       // Desired PORT ex 80
      // u_long interface, // Address to bind to.
      int backlog     // Number of pending connections in the backlog
      );

Public Methods

  • int connect_to_network(int t_socket_fd, struct sockaddr_in6 t_address) - Overrides the base class method. Connects the BindingSocket to the network.
  • void start_listening() - Starts listening on the BindingSocket.
  • void activate() - Activates the BindingSocket for incoming connections.
  • const int& get_back_log() - Returns the backlog value.
  • const int& get_listener() - Returns the listener file descriptor.
