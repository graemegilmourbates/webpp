Router Documentation

Class: Route

The Route class represents a route template and provides functionality for comparing routes and extracting URL parameters.

Constructor

      Route(std::string t_route);

Public Methods

  • bool compare(std::string &t_route) - Compares a real route against the route template.
  • URL_PARAMETERS get_parameters(std::string &t_route) - Returns the URL parameters extracted from the route.

Class: Router

The Router class handles routing and provides functionality for adding routes and handling requests.

Public Methods

  • void add_route(std::string t_route, ROUTE_HANDLER *t_handler) - Adds a route and its corresponding handler to the router.
  • void handle_request(WEBPP::Responder t_responder, REQUEST &t_request) - Handles a request by matching the route and invoking the appropriate handler.

Type Definitions

  • URL_PARAMETERS - Type alias for an unordered map of URL parameters.
  • URL_PARAMETER - Type alias for a pair of an integer (location) and a string (parameter value).
  • ROUTE_HANDLER - Type alias for the route handler function signature.
